# TRAZO

## Deploy (Render + Netlify)

Variables recomendadas para Render:

- `NODE_ENV=production`
- `NODE_VERSION=20.17.0`
- `TRUST_PROXY=1`
- `COOKIE_SECURE=1`
- `APP_ORIGIN=https://trazooo.netlify.app`
- `ALLOWED_ORIGINS=https://trazooo.netlify.app`
- `APP_DATA_ROOT=/tmp/trazo`
- `SUPABASE_URL=...`
- `SUPABASE_SERVICE_ROLE_KEY=...`
- `SUPABASE_BUCKET=trazo-tutorials`
